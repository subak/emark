Rounded Images

To create a rounded image simple wrap your image tag in a div with the class .img. The wrapping div will automatically adjust to the width and height of your image.

<div class="img">
	<img src="./img/avatar.png" alt="Avatar" />
</div>

You can also set the width of your rounded image via css: 

<style>
.img {
	width: 50px;
	height: 50px;
}
</style>

Pricing Plans

Plans Wrapper

All pricing plans need to be placed within this container.

<div class="pricing-plans plans-4">

	--- Plans go here ---
	
</div>

You can add or remove the number of pricing tables in a row using the following classes:

  • plans-1
  • plans-2
  • plans-3
  • plans-4

Faq Plugin

Faq Html

The html for the faq plugin is very basic. Styling and layout will be added when the plugin is called.

NOTE: Be sure the set your faq title as an h4 tag as that is what the table of contents is generated from.

<ol class="faq-list">
						
	<li>
		
		<h4>How can I get my own domain name?</h4>
		<p>The Internet Corporation for Assigned Names and Numbers (ICANN) maintains a list of...</p>		
	</li>
	
</ol>

Faq Javascript

Below is the javascript to initialize the faq plugin and the available configuration options

$('.faq-list').goFaq ({ 
	enableSearch: true,
	enableToc: true,
	enableStyling: true
});
